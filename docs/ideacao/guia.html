Hierarquia visual

Três níveis de superfície (ink-950 → 900 → 850) criam profundidade sem cor. Tamanhos tipográficos escalonados guiam o olhar: eyebrow → título → corpo.

Contraste

Texto base zinc-200 sobre fundo ink-950 (AAA). Acento brand-400 reservado para CTAs e destaques de preço.

Espaçamento

Grade de 4px. Cards p-4/5, seções py-8/12, container max-w-7xl com padding responsivo.
